安裝：

1.安裝環境 PHP 4.4 以上（無需mysql，ZIP操作需要zlib庫支持，縮略圖顯示需要GB庫支持）
2.確保以下文件為可寫
  class/group.php
  class/users.php
  info.php

PHPCMS文件管理器 功能特點：

1.可以遠程管理服務器上的文件，包括上傳,下載,新建,編輯,複製,粘貼,刪除,遠程下載等文件操作。 
2.採用 PHP + AJAX 的方式構建。客戶端採用緩存技術，不僅操作迅速，而且能大大減輕服務器的負擔。 
3.實現了遠程 ZIP文件打包/解壓。 
4.多用戶，權限管理，可以分配用戶各自的操作目錄和可操作的文件類型。 
5.兩種瀏覽方式:列表模式和圖標模式。 
6.在圖標模式下可以預覽圖片文件的縮略圖。 
7.強大的編輯器可以實現自動縮進，顯示行號 和 字符替換等功能。 
8.自定義風格模板。內置6套模板。
9.集成BlueShow圖片瀏覽器 
10.集成在線文件閱讀器 
11.Web2.0體驗 
12.兼容IE,Firefox,Opera等主流瀏覽器
