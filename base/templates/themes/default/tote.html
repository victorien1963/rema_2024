減塑生活：一只改變習慣的手提袋

環境保護理念是近代不斷被探討的議題，為了達到減少一次性塑膠物品的使用，我們不斷大量製造替代產品，如環保餐具、環保杯、購物袋、杯袋等，反而造成更多的資源浪費。當我們正以為透過「 減塑 」協助解決氣候變遷、海洋垃圾等兩大全球環境災難時，卻意外製造更多的危害。

我們停下腳步試想，「 真的有製作環保袋的必要嗎？」經過不斷的會議討論和縝密周全的思考，答案是肯定的。但 Rema 不想要只做出一個「 有時效性 」的環保袋，使用幾次之後就被存放在衣櫃深處，成為一只乏人問津的廢棄物，下場與一次性塑膠產品相同。我們更希望打造出的產品是能夠改變人類行為的媒介，如同堅守著經由著重巧思的機能服飾改寫您的運動習慣和生活日常的宗旨。

因此，我們經過長時間的研發設計和調整，參考市面上所有環保手提袋的材質、大小、耐用度，進而發展出三款多功能環保手提袋，採用紮實堅固的帆布與尼龍，結合寬敞方正的簡約外型，希望不只能有效減少一次性購物袋的使用，更希望打造一款經典且耐用的手提袋，陪伴所有消費者度過日常，改變既有的消費習慣，而真正達到永續環保精神。

環保彩繪托特包

Rema 特邀知名藝術家 Croter Illustration & Design Studio 所繪製的專屬亮眼彩繪，以品牌標誌結合機械貓咪圖案作為主要圖樣，構築出趣味可愛的樣貌。

環保字母托特包

汲取 Rema 品牌字母和創立年份為靈感，以簡潔俐落的輪廓體現品牌所追求的精煉精神，結合上經典勵志語錄，激發您每日的正能量，為日常生活打氣。

環保簡約托特包

極簡藝術的極致之美在於恰到好處的描繪經典，以品牌標示“ R ”字樣為主要視覺，不只象徵的 Rema 精神，堅守 Recycle ( 回收 )、 Reduce ( 減量 )、 Reuse ( 再造 )、 Repurpose ( 重生 ) 的精神，更代表的是 Reform ( 改革 )、 Redefine ( 重新定義 )，期望能為未來的運動和生活帶來革新與轉變。

我們之所以製作環保托特包不只是想要逐步走向微量塑料的道路前進，更是希望打造一只耐用且能陪伴您生活的隨身物品，藉由每一個人行為的改變，累積成為巨大能量，為自然環境維護產生更實質的效果。
